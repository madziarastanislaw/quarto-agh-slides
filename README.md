# quarto-agh-slides
[AGH](https://www.agh.edu.pl/en) slides template for Quarto

To use it, grab the [example `.ipynb` file](https://raw.githubusercontent.com/habemus-python/quarto-agh-slides/refs/heads/main/slides.ipynb) and edit it with your favourite Jupyter notebook tool.

To generate HTML files for presentation, do `quarto render defence.ipynb --execute --to revealjs`, the output should look like this:

<p align="center">
    <img 
        src="https://raw.githubusercontent.com/habemus-python/quarto-agh-slides/refs/heads/main/slides.gif" 
        alt="slides converted to animated gif"
        width="50%"
    />
</p>

To convert it to a pdf, open in a browser, select `☰ :: Tools :: PDF Export Mode` and use the browser's print dialog to save a pdf.
